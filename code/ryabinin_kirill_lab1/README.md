## Задание

Разработать программу, которая позволяет выполнять:

1. Точечную эластичность спроса по цене
2. Дуговая эластичность спроса по цене
3. Тип спроса
4. Эластичность спроса по доходу
5. Перекрёстная эластичность спроса
